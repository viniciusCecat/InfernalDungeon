# Banco de Dados

O projeto usa SQLite com SQL.js para salvar dados da conta e da loja.

Arquivo criado ao rodar o projeto:

```text
codigo-fonte/server/data/infernal-dungeon.sqlite
```

Tabelas:

- `users`
- `products`
- `addresses`
- `orders`
- `order_items`

Visualização no navegador:

```text
http://127.0.0.1:5173/api/database
```
