## Descricao resumida

Infernal Dungeon e uma wiki/apresentacao oficial do jogo proposto na Etapa 1. O jogo e um tower defense com tema de dungeon em que o jogador controla um nucleo, posiciona torres, recruta NPCs, evolui recursos em camadas mais profundas e invade outras dungeons controladas por NPCs. O site organiza as informacoes principais do universo, personagens/unidades, sistema de mascaras, atualizacoes e loja conceitual.

O fluxo funcional completo implementado e o gerenciamento de mascaras da wiki. O usuario consegue cadastrar uma mascara, consultar a lista, pesquisar/filtrar, editar dados e excluir registros. Os dados ficam salvos no `localStorage` do navegador.

## Instrucoes para execucao

1. Acesse a pasta do codigo:

```bash
cd InfernalDungeon/codigo-fonte
```

2. Instale as dependencias:

```bash
npm install
```

3. Rode o projeto:

```bash
npm run dev
```

4. Abra o endereco exibido pelo Vite no navegador, normalmente:

```text
http://localhost:5173
```

## Funcionalidades implementadas

- Pagina inicial de apresentacao do jogo.
- Secao de background/universo do Infernal Dungeon.
- Catalogo de personagens e unidades com filtro por categoria.
- Sistema de mascaras com cadastro, listagem, busca, filtro, edicao e exclusao.
- Secao de atualizacoes/devlog.
- Loja conceitual com produtos previstos.
- Interface componentizada em React.
- Persistencia local das mascaras usando `localStorage`.
